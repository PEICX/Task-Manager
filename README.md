# Task-Manager


## Intro
PyQt5 写的一个任务管理器，只是一个玩具；

Win10 和 Kali linux下测试没问题。

![image](https://github.com/PEICX/Task_Manager/blob/master/img/image1.png)

### project structure

```
|- OS.py         // 主文件
|- UI_OS.py      // 界面文件
|- UI_os.ui      // 界面文件原文件
```

## Begin

### Environment

```
Kali linux 64位
Python 3.6
PyQt5
```

### Usage

```
git clone git@github.com:PEICX/Task_Manager.git
cd Task_Manager
python OS.py
```
